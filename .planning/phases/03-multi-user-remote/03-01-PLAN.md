Implement SSH tunneling for secure remote DuckBrain access.

Purpose: Enable users to connect to remote DuckBrain instances via SSH, automatically creating a secure tunnel that forwards the remote HTTP server to a local Unix socket. This provides transparent remote access without opening firewall ports.

Output: Working SSH connection with dynamic port forwarding and multi-server support via named sockets.
</objective>
